---
title: Grouping
description: Group data by property values with configurable display modes, sorting, and URL persistence.
---

Grouping organizes your data into collapsible sections based on property values. Each view type supports grouping with accordion-style expansion.

## Groupable Property Types

Not all property types support grouping. The following types can be used with `groupBy`:

| Type | Supports Grouping | ShowAs Options |
|------|-------------------|----------------|
| `select` | Yes | — |
| `multiSelect` | Yes | — |
| `status` | Yes | `option`, `group` |
| `date` | Yes | `day`, `week`, `month`, `year`, `relative` |
| `number` | Yes | Range configuration |
| `text` | Yes | `exact`, `alphabetical` |
| `checkbox` | Yes | — |
| `formula` | No | — |
| `button` | No | — |
| `filesMedia` | No | — |

---

## Group Configuration

All group configurations share these common options:

```tsx
interface GroupOptions {
  hideEmpty?: boolean;   // Hide groups with no items (default: false)
  showCount?: boolean;   // Show item count in headers (default: true)
  sort?: "asc" | "desc"; // Sort groups (default: "asc")
}
```

## ShowAs by Property Type

### Status

Status properties can group by individual options or by their parent group.

```tsx
{
  propertyId: "availability",
  propertyType: "status",
  showAs: "option",  // or "group"
}
```

| ShowAs | Description |
|--------|-------------|
| `option` | Groups by individual status value (e.g., "In Stock", "Low Stock") |
| `group` | Groups by status group label (e.g., "Available", "Warning") |

**Example with `showAs: "option"`:**
```
▼ In Stock (12)
▼ Low Stock (3)
▼ Out of Stock (5)
```

**Example with `showAs: "group"`:**
```
▼ Available (15)    ← Contains "In Stock" + "Low Stock"
▼ Unavailable (5)   ← Contains "Out of Stock"
```

---

### Date

Date properties support multiple granularity levels for grouping.

```tsx
{
  propertyId: "createdAt",
  propertyType: "date",
  showAs: "month",           // Required for date
  startWeekOn: "monday",     // Optional, for "week" showAs
}
```

| ShowAs | Format | Example |
|--------|--------|---------|
| `day` | `Mon DD, YYYY` | "Mar 15, 2024" |
| `week` | `Mon DD - Mon DD` | "Mar 11 - Mar 17" |
| `month` | `Mon YYYY` | "Mar 2024" |
| `year` | `YYYY` | "2024" |
| `relative` | Smart buckets | "Today", "Yesterday", "Last 7 days" |

**Relative date buckets:**
- Today
- Yesterday
- Last 7 days
- Last 30 days
- Older months (e.g., "Feb 2024")
- Tomorrow
- Next 7 days
- Next 30 days
- Future months (e.g., "Apr 2024")

**Week start configuration:**

```tsx
{
  propertyId: "createdAt",
  propertyType: "date",
  showAs: "week",
  startWeekOn: "sunday",  // "monday" | "sunday"
}
```

### Number

Number properties group values into ranges using a `numberRange` configuration.

```tsx
{
  propertyId: "price",
  propertyType: "number",
  numberRange: {
    range: [0, 1000],  // Min and max values
    step: 100,         // Bucket size
  },
}
```

**Generated groups:**
```
▼ < 0 (2)
▼ 0 - 100 (15)
▼ 100 - 200 (8)
▼ 200 - 300 (12)
...
▼ 900 - 1000 (3)
▼ 1000+ (5)
```

Values below the range minimum go to `< {min}`, values above go to `{max}+`.

---

### Text

Text properties can group by exact value or alphabetically.

```tsx
{
  propertyId: "category",
  propertyType: "text",
  showAs: "alphabetical",  // or "exact"
}
```

| ShowAs | Description |
|--------|-------------|
| `exact` | Groups by exact text value (default) |
| `alphabetical` | Groups by first letter (A-Z, # for special characters) |

**Example with `showAs: "alphabetical"`:**
```
▼ A (5)
▼ B (3)
▼ C (8)
...
▼ # (2)   ← Numbers and special characters
```

---

### Select / MultiSelect

Select properties group by the selected option value.

```tsx
{
  propertyId: "category",
  propertyType: "select",
}
```

For `multiSelect`, items with multiple values appear in each matching group.

---

### Checkbox

Checkbox properties group into two buckets: checked and unchecked.

```tsx
{
  propertyId: "isActive",
  propertyType: "checkbox",
}
```

**Generated groups:**
```
▼ true (25)
▼ false (10)
```
